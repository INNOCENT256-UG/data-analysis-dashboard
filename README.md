HR Dataset — Data Cleaning & Analysis
Cleaned a raw HR dataset of 1,470 employee records using Python (Pandas) and Excel. Removed 214 duplicate and null rows, standardized formatting, and built an interactive dashboard showing salary distribution by department and employee attrition rates by age group. Tools used: Python, Pandas, Excel, Power BI.
